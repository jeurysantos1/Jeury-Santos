Approach

How I work.

01

Decide the variables separately.

I isolate each question, weigh it against real principles (Fitts, Norman, Jakob), then decide deliberately instead of by default.

02

Convention informs; the team decides.

I bring the platform research — what FB, IG, and iOS actually do — so trade-offs are understood, not assumed. The analysis earns the decision; it doesn't override it.

03

Prototype the seams.

The handoffs are where products break. I build the full flow end to end — backdrops, state machines, error states — so eng and content align on real behavior, not static comps.

04

Keep state truthful.

State should always tell the user exactly where they are — and pull them back to finish what they started.

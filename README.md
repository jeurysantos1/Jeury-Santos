# Jeury Santos Avila — Portfolio

Personal portfolio site for **Jeury Santos Avila**, Product Designer at Meta (Ads Manager App / BizComm).

Dramatic dark, Apple-style landing page with cinematic motion — built as a single, dependency-free `index.html` (only external dep: Google Fonts).

## Structure

```
index.html                              ← the homepage (this landing page)
case-study/
  index.html                            ← AMA Contact Collection case study
  mobilesite-case-study/index.html      ← MobileSite Contact Collection case study
agent/
  index.html                            ← jeury.agent widget demo (previous root project)
```

## Highlights

- **Signature palette:** electric teal `#1de9b6` + warm amber `#ffb454` on near-black `#06080b`.
- **Type:** Space Grotesk (display) + Inter (body).
- **Motion:** hero line-reveal, scroll parallax, per-card clip-path entrance, and a **click-to-enter** full-screen transition into each case study. Fully respects `prefers-reduced-motion`.
- **Two case studies** framed as two halves of one advertiser-reachability story.

## Preview

Open `index.html` in any modern browser. No build step.
